Hume input/output files

1       Input-files

1.1      Control file

Purpose

Bundling a set of single simulations to one set, enabling common statistical evaluation and parameter optimization

Contains:

A list of Control Ini-Filenames (see below), with complete path, one filename per row

Extension

 *.fn

Example name

Q:\WHEAT\GW_Scenarios\Inis\Uelzen\LT_Scenario_Uelzen_1972_2019.fn

Content Example

Q:\WHEAT\GW_Scenarios\Inis\Uelzen\LT_Scenario_Uelzen_1972.ini

Q:\WHEAT\GW_Scenarios\Inis\Uelzen\LT_Scenario_Uelzen_1973.ini

Q:\WHEAT\GW_Scenarios\Inis\Uelzen\LT_Scenario_Uelzen_1974.ini

Q:\WHEAT\GW_Scenarios\Inis\Uelzen\LT_Scenario_Uelzen_1975.ini

 

1.2      “Ini-Files”

Windows Ini-Files are used to store and retrieve values and options within HUME

1.2.1   Control-Inifile

Purpose

Storage and retrieval of basic information for initialization of a single model run

Contains:

Several sections (TimeInit, FileNames, MeasurementFiles) with information about Simulation time and filenames

Extension

 *.ini

Examplename

Q:\WHEAT\GW_Scenarios\Inis\Uelzen\LT_Scenario_Uelzen_1972.ini

Content Example

[TimeInit]

Startzeit=26177

Endzeit=26542

TimeStep=1

 

[FileNames]

ParamIniFN=Q:\WHEAT\GW_Scenarios\Inis\Uelzen\PAR_LT_Scenario_Uelzen_1972.ini

WeatherFileFN=C:\Users\...\HUME_WEATHER_Station_Uelzen_05146.txt

StateIniFN=Q:\WHEAT\GW_Scenarios\Inis\Uelzen\STATE_LT_Scenario_Uelzen_1972.ini

OptionsIniFN=Q:\WHEAT\GW_Scenarios\Inis\Uelzen\Options_LT_Scenario_Uelzen.ini

OutPutDir=P:\GW_Scenario\Uelzen

[MeasurementFiles]

1.5      Settings file

1.5.1   Properties.ini

Purpose

Storage and retrieval of values for controlling behavior of user interface (FormMod)

Contains:

Several sections (SectionName=Submodelname) with names and setting values of Hume Entities

Extension

 *.ini

Examplename

Q:\WHEAT\Project\Properties.ini

Content Example

[Files]

ControlFile=Q:\Wheat\INI\Tages\FN_Paper3\Log_103_13_14_BS_14_MER.fn

 

[SoilWaterModelR1]

act_evap.PlotTograpH=0

ActTrans.WriteFinalValue=0

ActTrans.WriteToFile=1

dt_int.PlotTograpH=0

dt_int.WriteFinalValue=0

dt_int.WriteToFile=1

 

2       Outputfiles

2.1      Settings documentation file (*_res.ini)

Purpose

Saves all settings for each single simulation file

Contains:

Several sections (SectionName=Submodelname) with names and setting values of Paremters etc.

Extension

 *_res.ini

Examplename

Q:\WHEAT\Project\”Initfilename”_res.ini

Content Example

[SimulationRun]

iniFile=Q:\WHEAT\GW_Scenarios\Inis\Soltau\LT_Scenario_Soltau__1972.ini

TimeOfRun=02.09.2019 15:53:03

 

[MeasurementFiles]

Development1=P:\GW_Scenario\Soltau\LT_Scenario_Soltau__1972_Development1_dat.csv
