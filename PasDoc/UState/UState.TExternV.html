Declaration

type TExternV = class(THumeNumEntity)

Description

TExternV is a class representing an external variable in the model. Values of model elements outside a submodel are called "external variables". They can be made accessible by adressing a pointer for that value. The default method for setting the link to the external variable is to search for it in the model by its name. It extends THumeNumEntity and provides methods for getting and setting the value, as well as creating the external variable with a name, units, and an external value type.
