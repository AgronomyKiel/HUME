Unit

UMod

Declaration

type TMod = class(TGraphicControl)

Description

Base type for the central model component It summarizes the submodels of the simulation model and provides methods for data output, parameter estimation, and sensitivity analysis. It is descended either from Tobject if compiled for command line application or from TPersistent if compiled for GUI appplication.
