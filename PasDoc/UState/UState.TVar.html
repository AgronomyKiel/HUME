Unit

UState

Declaration

type TVar = class(THumeNumEntity)

Description

TVar is a class representing a variable, i.e. a value that can be calculated for each time step in the model from other variables, state variables or parameters. It extends THumeNumEntity and provides methods for self writing of source code for its initialization, rate calculation.
