Declaration

type TRasterData = class(TObject)

Description

Declaration of class TRasterData. This class encapsulates the raster, positions and the number of read and randomized data (WAP). It enables display in the tabs RasterData and RootDistribution. The class also handles file access.

procedure readRasterData(fn: string; var Series: TPointSeries);

Reads aggregated raster data (root counts) which are then used to randomly determine coordinate points. Example of functionality: readln(f, Ncols) assigns the value read from f to the variable Ncols.

procedure readXYfromFile(Filename: TFilename; var Series: TPointSeries); virtual;

Reads XY coordinates from a file and successively fills the passed (call-by-reference) PointSeries object with the XY pairs.

procedure saveRootPositions(SaveDialog: TSaveDialog);

After reading aggregated root data, saves the generated random root coordinates (considering assignment to a 5x5 cm grid cell) as floating point numbers in a file.
