Overview

Classes, Interfaces, Objects and Records

Name Description
Class TBaseSubmodRootDiff Declaration of class TSubmodRootDiff. Base class for derived diffusion models Functionality of the base class: all models can read raster data, exclude margins from calculations and display the data in the appropriate tabs of the Hume form. The 1D model does not output in the 3D plot tab.
Class TPointDoubleType Array storing TPointDoubleTypes used to read xy coordinates from a file. The record extends the type posi from the original model by the fields root, xi, yi, NInflux, WInflux and area.
Class TRasterData Declaration of class TRasterData. This class encapsulates the raster, positions and the number of read and randomized data (WAP). It enables display in the tabs RasterData and RootDistribution. The class also handles file access.
Class TRootPosition Defines an object for storing the information of a single root object x and y are the floating point type coordinate xi and yi are the index integers for the cell of a 2-D grid object the root object is located
Class TSRPLight Lean, memory-saving version of an SRP used in the 2d diffusion model for interface with the structural root model TSRP instances of the single root model also need fields for area and the vertex list so that, when reading raster data, the surface area can be determined (using Voronoi polygons or alternatively by splitting the area of raster cells among the roots contained in them. Units may be an issue)
