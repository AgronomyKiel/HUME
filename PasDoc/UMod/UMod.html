Description

UMod is a unit that defines the base classes TMod and TSubmodel. TMod can be instantiated directly and is the central control object of a simulation model. It summarizes the submodels of the simulation model and provides methods for data output, parameter estimation, and sensitivity analysis. TSubModel is an abstract base class for problem specific submodels. Furthermore some other classes are defined.
