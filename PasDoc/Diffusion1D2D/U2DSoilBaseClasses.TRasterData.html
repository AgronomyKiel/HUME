Declaration

type TRasterData = class(TObject)

Description

Declaration of class TRasterData. This class encapsulates the raster, positions and the number of read and randomized data (WAP). It enables display in the tabs RasterData and RootDistribution. The class also handles file access.

procedure CalcEqualDistribution(RLD:real);

All diffusion models can compute a uniform distribution from a given one. The method distributes points based on a given WLD evenly over an area. The TPointSeries object of the calling HUME GUI method is refilled so the display can be updated. Different methods were created for this purpose and can be switched here.

procedure distributeHexagonCol;

Distribution using hexagons. Columns are filled sequentially. The method may be advantageous when nutrient uptake in layers is calculated (this still needs verification). The distribution was adjusted: X-values were shifted half a Rad_AK to the right and Y-values half a Rad_AK upward (see slide Filling columns 2). CAVE: method not up to date and should be improved based on distributeHexagonRow (e.g., calculation of ErrorReg).

procedure distributeHexagonRow;

Distribution using hexagons. Rows are filled sequentially. For better distribution the points still need to be shifted by half the radius of the circumscribed circle.

procedure WriteOutputToFile(fn:string; Time:real); virtual;

Writes coordinates and areas of PosArr to a file. With a structural model, output occurs at each time step; with static root positions only at the start of the model run. For testing in the static case, the positions of roots in the margins are also written.
