Declaration

type TGrowthCurvePlant = class(TAbstractPlant)

Description

Plant model that simulates growth curves based on logistic, monomolecular, Gompertz, or Richards growth. It calculates growth rates based on temperature and other parameters, and integrates these rates over time to update state variables such as LAI, DM, CropHeight, and ShootN.

This class extends TAbstractPlant and provides methods for calculating growth rates, integrating state variables, and handling temperature effects on plant growth. It also supports soil nitrogen uptake if enabled.
